# Abje-Waves

渲染[全てあなたの所為です。](https://www.youtube.com/@subeteanatanoseidesu)的作品《[アブジェ](https://www.youtube.com/watch?v=ORz2SsyIGgk)》中的波纹

![示例](docs/img1.png)

可以从字符串输入，编码波纹的属性

例如：`EncodeType.SIDE: "でぐちはすぐそこ"` →渲染为 `10100100...`

## Getting Started

```bash
pip install uv
uv sync
uv run main.py
```

打开`main.py`与`config.py`查看更多可配置项

## Output

- `format`参数支持 `mp4 `、`mov `、`webm`
  - 选用 `mov`格式可支持透明度输出
